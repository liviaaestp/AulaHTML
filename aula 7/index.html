Aula 07 - Estudo de CSS

A sigla CSS é Cascading Style Sheets é uma linguagem ultilizar para ultilizar e/ou diagramar elementos html de uma página web

O CSS funciona em conjunto com o HTML, enquanto HTML diz o que será exibido nas áginas, o CS diz como severá ser exibido.
